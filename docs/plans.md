# Plans

## Source
- Task: разработать production-ready WinForms приложение для администрирования склада по требованиям `README.md`
- Canonical input: `README.md` и пользовательский запрос от 2026-04-09
- Repo context: весь репозиторий
- Last updated: 2026-04-09

## Assumptions
- Используется `net8.0-windows` как стабильная LTS-платформа с поддержкой WinForms и совместимостью с установленным runtime.
- Архитектура строится как desktop monolith с разделением на слои `Domain`, `Application`, `Infrastructure`, `Presentation`.
- Локальное хранение реализуется через SQLite и EF Core для ускорения разработки и предсказуемой миграции схемы.
- Аутентификация не требуется; единственная рабочая роль `Кладовщик` задается как системный контекст пользователя.
- Печать форм ИНВ-3/ИНВ-19 и упаковочных листов реализуется как генерация печатного текста/preview внутри приложения без внешних интеграций.

## Validation Assumptions
- Проверка качества выполняется командами `dotnet restore`, `dotnet build`, `dotnet test`.
- Для UI smoke-проверок используются сборка, запуск стартап-проекта и проверка ключевых экранов на seeded данных.
